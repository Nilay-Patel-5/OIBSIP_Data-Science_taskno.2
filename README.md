# OIBSIP_Data-Science_taskno.2
# Unemployment Analysis with Python

## 📌 Project Overview

This project was completed as part of the **Oasis Infobyte Data Science Internship (Task 2)**.

The objective of this project is to analyze unemployment trends in India using Python. The analysis includes data exploration, data cleaning, visualization, and extraction of meaningful insights from unemployment data.

The project helps understand unemployment patterns across different states, urban and rural areas, and monthly trends.

---

## 📊 Dataset Information

Dataset: **Unemployment in India**

Source:
https://www.kaggle.com/datasets/gokulrajkmv/unemployment-in-india

The dataset contains unemployment statistics across various Indian states and regions.

### Features

* Region
* Date
* Frequency
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)
* Area (Urban/Rural)

## 📂 Project Workflow

### Step 1: Import Libraries

Required libraries such as Pandas, Matplotlib, and Seaborn are imported for data analysis and visualization.

### Step 2: Load Dataset

The CSV dataset is loaded into a Pandas DataFrame.

### Step 3: Data Exploration

The following operations are performed:

* View first few records
* Check dataset structure
* Identify missing values
* Generate statistical summary

### Step 4: Data Cleaning

* Remove unwanted spaces from column names
* Convert Date column into datetime format

### Step 5: Data Visualization

Several visualizations are created:

#### 1. Distribution of Unemployment Rate

Histogram showing the frequency distribution of unemployment rates.

#### 2. State-wise Average Unemployment Rate

Bar chart displaying average unemployment rates for different states.

#### 3. Urban vs Rural Unemployment

Boxplot comparing unemployment rates in urban and rural regions.

#### 4. Monthly Unemployment Trend

Line chart illustrating unemployment trends over different months.

#### 5. Top 10 States with Highest Unemployment

Horizontal bar chart highlighting states with the highest unemployment rates.

#### 6. Correlation Heatmap

Heatmap showing relationships between numerical variables.
